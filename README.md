# SDL2 Color Picker

## Requirements
Requires [SDL_ttf](https://www.libsdl.org/projects/SDL_ttf/release-1.2.html)
```bash
sudo apt install libsdl-ttf2.0-dev
```
SDL

## Running
```
make
./colorSelector
```
